# 🎓 CET College Eligibility App

## 📌 Project Description
This app helps students:
- Enter CET rank and category
- Upload cutoff CSV
- Get list of eligible colleges
- Chat about college using Groq AI
- Download eligible colleges as CSV

## 🚀 Features
- CSV Upload
- Rank-based filtering
- Category-wise cutoff check
- Groq AI Chat Assistant
- CSV Download
